# pyMCP-Lab
This repo is a companion for [Ship an MCP Server in Python - Fast](https://www.packtpub.com/en-us/product/ship-an-mcp-server-in-python-fast-9781807609085) by Christoffer Noring, published by Packt Publishing.

I was able to follow along through chapter 7 (section 8) and "Converting tools for the LLM".

After that, the book leaves off from being a step-by-step guide and the reader is left to their own devices.
